Welcome to the **Acid Lounge Design System** - a component library for the Shaken Not Stirred party game platform.

## Tech Stack

- **React 18** - UI framework
- **Storybook 8** - Component documentation and development
- **Pixi.js 8** - WebGL graphics and effects
- **Vite 6** - Build tool
- **Turborepo** - Monorepo management

## Design Philosophy

This design system follows **Storybook Driven Development (SDD)** - all visual components are built and approved in Storybook before being integrated into the application logic.

### Atomic Design

Components are organized using Brad Frost's Atomic Design methodology:

- **Atoms** - Basic building blocks (Button, Input, Typography, Avatar)
- **Molecules** - Simple combinations (PlayerCard, Timer, FormField)
- **Organisms** - Complex sections (SplashScreen, Lobby, GameBoard)
- **Templates** - Page layouts

## Visual Identity: Acid Lounge

The "Acid Lounge" aesthetic combines:

- **Neon colors**: Acid Green (#CCFF00), Hot Pink (#FF10F0), Electric Purple (#9D00FF)
- **Dark backgrounds**: Pure black (#000000) optimized for TV displays
- **Kinetic typography**: Bebas Neue for impact, Orbitron for futuristic feel
- **Liquid animations**: Morphing blobs and psychedelic effects
- **Glow effects**: Multi-layer neon shadows

## Getting Started

```bash

pnpm install

# Start Storybook

```
